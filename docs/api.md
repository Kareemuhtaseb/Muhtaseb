# API Endpoints

- `GET /api/tenants`
- `POST /api/tenants`
- `GET /api/units`
- `GET /api/payments`
- `GET /api/expenses`
- `GET /api/owners`
- `GET /api/distributions`
- `GET /api/net-income` – total rent minus expenses
- `GET /api/distributions/calculate` – suggested payouts per owner
- `GET /api/reports/monthly` – payments and expenses grouped by month
